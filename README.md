# Portfolio Wadia

Portfolio personal desarrollado con HTML y CSS.

## Archivos principales

- `index.html`: versión inicial del portfolio.
- `portfolio_wadia_mejorado.html`: versión mejorada.
- `portfolio_wadia_corregido.html`: versión revisada.
- `PortfolioWadia.html`: versión alternativa conservada como referencia.
- `images/`: recursos gráficos del portfolio.

El proyecto funciona directamente en el navegador, sin backend ni dependencias.
